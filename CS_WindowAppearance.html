アピアランスの取得方法

Win32APIを呼び出すしかない模様。

フレームの幅はSM_CXFRAME, SM_CYFRAME にて取得できる。試してみた所、ビルド環境の .NET Framework のバージョンでも変化する模様。
調べたところ.NET Framework 4.5 以降から違ってきている。

.NET Framework 3.0      8, 8
.NET Framework 3.5      8, 8
.NET Framework 4.0      8, 8
.NET Framework 4.5      4, 4
.NET Framework 4.6.1    4, 4

見た目上は変化なし。
